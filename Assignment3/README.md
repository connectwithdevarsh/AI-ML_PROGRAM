# Assignment 3: Advance Analytics Using Matplotlib

This project contains the solutions to 7 data analytics and visualization problems solved using basic Python and Matplotlib, mimicking the coding style of a second-year Diploma IT student.

## Python Version
- Python 3.x (tested on Python 3.9+)

## Required Library
- matplotlib

To install the required library, run:
```bash
pip install matplotlib
```

## How to Run
Run any of the Python files using the standard Python interpreter. For example:
```bash
python problem1.py
```
Each script will output the answers to the respective questions in the terminal and display a graph.

## Project Files
- **problem1.py**: Airline Passenger Load Growth Analysis
- **problem2.py**: Software Bug Resolution Time Analysis
- **problem3.py**: Smart City Water Consumption Analysis
- **problem4.py**: Factory Shift Power Usage Analysis
- **problem5.py**: Online Learning Session Duration Analysis
- **problem6.py**: E-Commerce Order Fulfillment Growth Analysis
- **problem7.py**: Student Exam Results Analysis
