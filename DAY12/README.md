# Day 12 Internship - Used Car Valuation

## Project Name
Day 12 Used Car Valuation Portal

## Python Version
Python 3.x

## Libraries Used
- pandas
- streamlit
- scikit-learn
- pickle

## Files Included
- `train_model.py` - Script to train the regression model.
- `app.py` - Streamlit web application.
- `car_model.pkl` - Trained machine learning model (generated by training).
- `car_prices (1).csv` - Car prices dataset.

## How to Run

### Step 1
Run the training script to generate the model:
```bash
python train_model.py
```

### Step 2
Run the Streamlit application:
```bash
streamlit run app.py
```
