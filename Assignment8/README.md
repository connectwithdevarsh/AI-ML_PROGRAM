# ASGN8: Smart Surveillance & Image Management System

This project contains Python scripts to implement various components of a Smart Surveillance & Image Management System using OpenCV.

## Python Version
- Python 3.x

## Libraries
- OpenCV (`opencv-python`)

## Installation
```bash
pip install opencv-python
```
