# Assignment 5: Uber Dataset NCR Analysis

This project analyzes the Uber dataset NCR using Pandas and Matplotlib.

## Python Version
- Python 3.x (tested on Python 3.9+)

## Libraries Used
- pandas
- matplotlib
- openpyxl (to read Excel files)

## Dataset Name
- Uber data set ncr.xlsx

## How to Run
First, make sure the required libraries are installed:
```bash
pip install pandas matplotlib openpyxl
```

Then run the script:
```bash
python main.py
```
This will print output for Sections 1, 2, and 3 in the console and show the graphs from Section 4.
