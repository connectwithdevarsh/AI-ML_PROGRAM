# Day 9 Internship - Linear Regression

## Project Name
Day 9 Linear Regression Predictions

## Python Version
Python 3.x

## Libraries Used
- pandas
- matplotlib
- scikit-learn

## Files Included
- `advertising_sales_prediction.py` - Predicts sales units based on TV budget.
- `house_price_prediction.py` - Predicts house prices based on square feet.
- `advertising_sales.xlsx` - TV budget and sales dataset.
- `house_prices.xlsx` - House sizes and prices dataset.

## How to Run
First, install the required packages:
```bash
pip install -r requirements.txt
```

Run the scripts:
```bash
python advertising_sales_prediction.py
python house_price_prediction.py
```
