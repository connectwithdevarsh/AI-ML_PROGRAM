# Day 3 Internship - Matplotlib Tasks

## Project Name
Day 3 Matplotlib Visualization Tasks

## Python Version
Python 3.x (Recommended: Python 3.8 or higher)

## Library Used
- `matplotlib`

## Files Included
This directory contains 30 independent Python scripts:
- `problem1.py` to `problem10.py`: Categorical data visualization using Bar Charts.
- `problem11.py` to `problem20.py`: Sequential data visualization using Line Charts.
- `problem21.py` to `problem30.py`: Dictionary-based data visualization using Bar Charts.

## How to Run
To run any of the scripts, make sure you have `matplotlib` installed:
```bash
pip install matplotlib
```

Then, execute the desired script from the terminal:
```bash
python problem1.py
```
*(Replace `problem1.py` with the filename of any other problem you wish to run).*
