# Assignment 2: Analytics from Real Life Problems Using Data Visualization

This project solves 10 real-world analytical problems using basic Python code and data visualization with Matplotlib.

## Python Version
- Python 3.x (tested on Python 3.9+)

## Required Library
- matplotlib

To install the required library, run:
```bash
pip install matplotlib
```

## How to Run
Run any of the Python files using the standard Python interpreter. For example:
```bash
python problem1.py
```
Each script will output the answers to the respective questions in the console and display a graph.

## File List
- **problem1.py**: E-Wallet Transaction Volume Analysis
- **problem2.py**: Manufacturing Quality Inspection Time Assessment
- **problem3.py**: Ride-Hailing Driver Availability Study
- **problem4.py**: Corporate Meeting Room Utilization Study
- **problem5.py**: Weather Station Rainfall Variability Study
- **problem6.py**: Public Bus Route Punctuality Evaluation
- **problem7.py**: Warehouse Cold Storage Temperature Control
- **problem8.py**: Insurance Claim Processing Time Assessment
- **problem9.py**: Corporate Email Server Response Monitoring
- **problem10.py**: Water Treatment Plant Output Stability Review
