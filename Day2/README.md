# Day 2 Internship

This project contains Python programs developed during Day 2 of the internship. The programs analyze different real-world scenarios and visualize the data using matplotlib.

## Python Version
Python 3.x

## Library Used
- `matplotlib`

## Files Included
- `problem1.py`: Advertising Spend vs Sales Conversion Analysis
- `problem2.py`: Monthly Revenue Comparison Across Branches
- `problem3.py`: Employee Skill Level vs Productivity Analysis
- `problem4.py`: Company Expense Distribution by Department
- `problem5.py`: Product Price vs Customer Rating Relationship
- `problem6.py`: Manufacturing Defect Rate vs Production Volume
- `problem7.py`: Hotel Room Occupancy Across Seasons
- `problem8.py`: Online Course Category Enrollment Share
- `problem9.py`: Daily Website Bounce Rate Trend
- `problem10.py`: Salesperson Experience vs Monthly Sales

## How to Run
You can run any of the python files directly using:
```bash
python Day2/problem1.py
```
