# Employee Salary Prediction Portal

## Python Version
Python 3.10+

## Libraries
- streamlit
- pandas
- numpy
- scikit-learn
- matplotlib

## Dataset
- employee_salary.csv

## Files
- train_model.py
- app.py
- salary_model.pkl
- requirements.txt
- README.md

## How To Run

### Step 1
Run the training script to train and save the model:
```bash
python train_model.py
```

### Step 2
Run the Streamlit application:
```bash
streamlit run app.py
```
