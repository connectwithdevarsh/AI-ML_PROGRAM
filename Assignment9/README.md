# Assignment 9: Machine Learning Predictions

This project implements machine learning models (Linear Regression) to predict water usage and parking revenue.

## Python Version
- Python 3.x (tested on Python 3.9+)

## Libraries Used
- pandas
- matplotlib
- scikit-learn
- openpyxl (to read Excel files)

## Dataset Used
- `water_usage.xlsx` (for water prediction)
- `parking_revenue.xlsx` (for parking prediction)

## How to Run
First, make sure the required packages are installed:
```bash
pip install pandas matplotlib scikit-learn openpyxl
```

Then run the scripts:
```bash
python water_prediction.py
python parking_prediction.py
```

## Files
- **water_prediction.py**: Predicts water usage for a given number of households.
- **parking_prediction.py**: Predicts parking revenue for a given number of vehicles.
