# Day 10 Internship - Random Forest Classification

## Project Name
Day 10 Random Forest Predictions

## Python Version
Python 3.x

## Libraries Used
- pandas
- scikit-learn

## Files Included
- `telecom_churn_prediction.py` - Predicts telecom churn status.
- `health_bmi_prediction.py` - Predicts health/BMI diagnostic outcome.
- `telecom_churn (1).xlsx` - Telecom customers dataset.
- `HEALTH BMI (1).xlsx` - Health diagnostics dataset.

## How to Run
First, install the required packages:
```bash
pip install -r requirements.txt
```

Run the scripts:
```bash
python telecom_churn_prediction.py
python health_bmi_prediction.py
```
